# test_tasks
## How to start
1. Install [node.js](https://nodejs.org)
2. Install [mocha](https://mochajs.org/#installation): 
```
$ npm install mocha 
```
3. Start test
```
$ npm test
```

## About files
* showTitle.js - general function 
* test/test.js - unit tests
* test/data.csv - data for unit tests
